# ResearchPortfolio-FastAPI

Python/FastAPI service for the Research Portfolio. Hosts interactive model and calculation endpoints so research demos run live rather than as static screenshots.

## Endpoints

| Method | Path | Auth | Purpose |
|---|---|---|---|
| GET | `/health` | — | Liveness probe |
| GET | `/hello` | Bearer JWT | Smoke-test / placeholder |
| POST | `/stats/sample` | API key | Statistical distribution sampler |

## Stack

- FastAPI on Python 3.10
- Gunicorn + Uvicorn workers (production), Uvicorn reload (development)
- Structured request-scoped logging with real-IP detection and request-ID tracing

## Environment variables

All secrets are generated by `utilities/nas/bootstrap_env.sh` in the deployment repo.

| Variable | Purpose |
|---|---|
| `RP_FASTAPI_JWT_SECRET` | Shared secret used to verify JWTs issued by the AuthAPI |
| `RP_ADMIN_TOKEN` | Token for admin-only operations; never sent to the browser |
| `RP_FASTAPI_API_KEY` | API key required by the stats and demo endpoints |
| `CORS_ALLOWED_ORIGINS` | Comma-separated list of allowed frontend origins |
| `DEBUG` | Set to any non-empty value to enable debug log level |

## Local development

```bash
python -m venv venv
source venv/bin/activate        # Windows: venv\Scripts\activate
pip install -r requirements.txt

# Copy values from your local .env (generated by bootstrap_env.sh)
export RP_FASTAPI_JWT_SECRET=...
export RP_ADMIN_TOKEN=...
export RP_FASTAPI_API_KEY=...
export CORS_ALLOWED_ORIGINS=http://localhost:4200

uvicorn src.main:app --reload --port 8000
```

## Docker

```bash
docker build -t researchportfolio-fastapi .
docker run --rm -p 8000:8000 \
  -e RP_FASTAPI_JWT_SECRET=... \
  -e RP_ADMIN_TOKEN=... \
  -e RP_FASTAPI_API_KEY=... \
  -e CORS_ALLOWED_ORIGINS=http://localhost:4200 \
  researchportfolio-fastapi
```

In production this runs via `docker-compose.yml` in the deployment repo alongside the AuthAPI, PostgreSQL, and Cloudflare Tunnel.
